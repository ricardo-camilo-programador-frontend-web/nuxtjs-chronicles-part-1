## 🚀 Tecnologias Implementadas

## 🛠️ Setup Local

```bash
# Instalação de dependências
pnpm install

# Servidor de desenvolvimento
pnpm dev

# Build para produção
pnpm build
```

## 🌍 Variáveis de Ambiente

```env
NUXT_PUBLIC_GTM_ID=
NUXT_PUBLIC_SITE_URL=
NUXT_PUBLIC_GOOGLE_SITE_VERIFICATION=
```
